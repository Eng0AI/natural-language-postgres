# Natural Language Postgres

A demo app that lets you ask questions in plain English and get answers from your database using AI-powered natural language to SQL conversion.

## Tech Stack

- Next.js 15
- React 19
- Vercel AI SDK
- OpenAI
- PostgreSQL
- Tailwind CSS
- Recharts
- Framer Motion
